## About

我关注 **AI x 教育场景** 的产品化实践，主要方向包括学习支持工具、教育数据分析、AI workflow 与前端原型开发。

常用工具包括 Codex、Claude Code、VS Code、Python、Coze、Netlify Drop、v0、Vercel。

我注重三个问题：

- AI 能否真正进入具体教育场景
- 产品是否形成清晰的使用闭环
- 技术是否能降低学习、教学或信息处理成本

### LearnLoop

LearnLoop 是一个面向学习者的 AI 辅助学习工具，帮助用户将碎片化知识高效组织并生成个性化学习路径。通过卡片式任务和进度追踪机制，用户可以管理学习内容，实现持续进步与核心知识点掌握。

### TimeLume

TimeLume 用自然语言记录时间投入，通过锚点、时间流与周期报告，帮助用户重新看见自己的时间。

### Idea Editor

Idea Editor 是一款创意发散与重组工具，帮助用户将原始想法拆解、组合和优化，快速生成可落地的创意方案。内置类比迁移、组合重组和观察洞察等策略，让灵感像文字一样被编辑和扩展。

### BackCurve

BackCurve 是一个 AI 脊柱体态自查原型，通过分析用户正面背影照片，检测双肩差值、骨盆倾斜及脊柱弯曲情况。产品目标是提供低门槛、可视化的体态自查工具，辅助日常健康管理。
